## 9. AI向け参照ガイド（ES作成時の指針）

> このセクションはAIがES作成時に参照します。編集不要です。

### 情報の優先順位

ESを作成する際、情報が重複している場合は以下の優先順位で参照すること:

1. **このファイル（人物像まとめ.md）** → 人物像の骨格・全体像（最優先）
2. **エピソード対応表.md** → どのエピソードをどの質問に使うか
3. **各ESテンプレートファイル** → 詳細・具体例
4. **企業フォルダ（企業/選考中/[企業名]/）** → 企業固有の情報・過去ES

### ES作成の手順

1. まずこのファイルで人物像の全体を把握する
2. `エピソード対応表.md` で質問に対応するエピソードを確認
3. 該当するテンプレートファイルで詳細を確認
4. 企業ごとのカスタマイズは企業フォルダを確認

### 文字数調整のルール

| 文字数 | 構成 |
|-------|-----|
| 100字以下 | 結論のみ |
| 200字 | 結論 + エピソード概要 |
| 400字 | 結論 + エピソード + 工夫・学び |
| 600字以上 | 結論 + エピソード + 工夫 + 結果 + 今後への活かし方 |

### 注意事項

- 同じESで同じエピソードを2回使い回さない
- 数値（期間、人数など）は各テンプレートファイルの記載を正とする
- 企業の特性に合わせて強調ポイントを変える（技術系→設計の話、コンサル系→チームワークの話）
- 抽象的な表現（コミュニケーション能力、論理的思考力）は使わず、具体的な行動で表現する

## ⚠️ 注意事項

1. **同じESで同じエピソードを2回使わない**
   - 自己PRとガクチカで同じエピソードを使う場合は、切り口を変える

2. **企業の特性に合わせてエピソードを選ぶ**
   - 技術系企業には開発経験、コンサル系にはチームワーク経験など

3. **文字数に応じて詳細度を調整**
   - 200字: 結論 + エピソード概要
   - 400字: 結論 + エピソード + 工夫・学び
   - 600字以上: 結論 + エピソード + 工夫 + 結果 + 今後への活かし方

4. **数値は正確に記載**
   - 期間、人数などは各参照ファイルの記載を確認すること

# ポートフォリオ（エピソード集）

このフォルダには、ESや面接で使用するエピソードを個別ファイルとして保存します。

---

## 📁 ファイルの作成方法

1. `_template.md` をコピー
2. ファイル名を `【エピソード名】.md` に変更
3. テンプレートに沿って内容を記入

---

## 📝 エピソードの種類

### 開発経験
- 個人開発プロジェクト
- チーム開発プロジェクト
- ハッカソン参加
- インターン経験

### 課外活動
- サークル・部活動
- イベント企画・運営
- ボランティア

### 実務経験
- アルバイト
- インターン
- 長期インターン

### 学業
- 研究活動
- ゼミ活動
- 資格取得

---

## 💡 記入のコツ

1. **数値を具体的に**
   - 期間、人数、成果などは具体的な数値で
   - 例: 「3ヶ月」「60名規模」「2年間」

2. **工夫した点を詳しく**
   - ESで最も重要な部分
   - 「何を」「なぜ」「どうやって」を明確に

3. **学びを言語化**
   - この経験から何を得たのか
   - 入社後にどう活かせるか

4. **複数の切り口を用意**
   - 同じエピソードでも、技術面・チームワーク面など複数の切り口で書けるように

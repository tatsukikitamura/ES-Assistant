# 就活の軸（面接用）

## 🎯 このファイルの用途
- **使用する場面**: 面接で「就活の軸は？」と聞かれたとき
- **時間目安**: 1〜2分

---

## 📌 軸の一覧

> 3〜4つの軸を簡潔にまとめてください。

1. **【軸1】**: 
2. **【軸2】**: 
3. **【軸3】**: 
4. **【軸4】**: （あれば）

---

## 💬 回答テンプレート

### 簡潔版（30秒）

```
私の就活の軸は3つあります。
1つ目は【軸1】です。
2つ目は【軸2】です。
3つ目は【軸3】です。
御社は【企業の特徴】という点で、私の軸に合致していると考えています。
```

### 詳細版（1分）

```
私の就活の軸は3つあります。

1つ目は【軸1】です。【理由を一言】。

2つ目は【軸2】です。【理由を一言】。

3つ目は【軸3】です。【理由を一言】。

御社は【企業の特徴1】や【企業の特徴2】という点で、
私の軸に合致していると考え、志望しています。
```

---

## 📝 補足メモ

- 軸は覚えておき、すぐに答えられるようにする
- 企業ごとに「御社は〜」の部分をカスタマイズする
- 軸の理由を聞かれたときのために、背景も準備しておく

# 自己紹介テンプレート

## 🎯 このファイルの用途
- **使用する場面**: 面接の冒頭、グループディスカッション、説明会など
- **時間目安**: 30秒〜2分

---

## 📌 自己紹介の構成

1. 名前・所属
2. 専攻・学んでいること
3. 課外活動・アルバイト
4. 趣味・特技（あれば）
5. 一言アピール

---

## 💬 自己紹介テンプレート

### 30秒版（面接冒頭など）

```
〇〇大学〇〇学部〇年の【名前】と申します。
大学では【専攻・学んでいること】を学んでいます。
課外活動では【サークル・アルバイトなど】に取り組んでいます。
本日はよろしくお願いいたします。
```

### 1分版（一般的な自己紹介）

```
〇〇大学〇〇学部〇年の【名前】と申します。
大学では【専攻・学んでいること】を学んでいます。

課外活動では【活動内容】に取り組んでおり、【簡単な成果・学び】を経験しました。

また、【アルバイト・インターン】では【簡単な内容】を行い、【学んだこと】を学びました。

趣味は【趣味】で、【一言コメント】。

本日はよろしくお願いいたします。
```

### 2分版（詳しい自己紹介）

```
〇〇大学〇〇学部〇年の【名前】と申します。
大学では【専攻・学んでいること】を専攻しており、【具体的に学んでいること】に興味を持って学んでいます。

課外活動では【サークル・部活】に所属しており、【役職・役割】として【具体的な活動内容】に取り組んできました。
この経験を通じて、【学んだこと】を学びました。

また、【アルバイト・インターン先】で【期間】【業務内容】を行っており、【学んだこと・身についたスキル】を身につけました。

趣味は【趣味】で、【エピソード・こだわり】。

私の強みは【強み】で、御社でも【どう活かすか】に貢献したいと考えています。

本日はよろしくお願いいたします。
```

---

## 📝 補足メモ

- 暗記するのではなく、要点を押さえて自然に話す
- 面接官の反応を見ながら、長さを調整する
- 笑顔とアイコンタクトを忘れずに
- 「本日はよろしくお願いいたします」で締める
